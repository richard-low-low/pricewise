# Chrome Web Store 商品详情（中文参考版）

## 扩展名称

PriceWise - Amazon Price Tracker

## 简短描述（132 字符以内）

Track Amazon prices privately. See price history charts, set drop alerts. No data collection, no affiliate hijacking.

## 详细描述

PriceWise 在亚马逊商品页面上直接显示价格历史图表，帮你追踪价格变化趋势。支持 9 个亚马逊站点（美国、英国、德国、法国、日本、加拿大、澳大利亚、意大利、西班牙），支持多种货币。

**核心功能：**

- 商品页面内嵌价格历史走势图
- 一目了然的最低价、最高价、平均价统计
- 降价提醒 -- 设定目标价格，降价时自动通知
- 支持 9 个亚马逊区域站点及对应货币
- 最长保存 2 年的个人价格追踪记录

**隐私至上，绝不妥协：**

- 所有价格数据存储在你的浏览器本地，不会离开你的设备
- 零数据收集，没有分析统计，没有行为追踪，没有遥测
- 不劫持联盟链接，PriceWise 绝不修改或注入任何联盟推广代码
- 最小权限原则，仅访问亚马逊商品页面

**为什么选择 PriceWise？**

主流价格追踪工具存在严重问题。Honey 被曝光劫持内容创作者的联盟链接，将推广代码替换为自己的。Keepa 需要注册账号，并且会收集用户浏览数据，存在隐私隐患。

PriceWise 采用完全不同的路线：数据留在你的设备上，代码完全开源，专注做好一件事 -- 透明地追踪价格。

**免费且开源。**

PriceWise 完全免费使用。源代码公开可查，任何人都可以审计或参与贡献。
